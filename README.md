# chanapp-gzq-h5 boilerplate
> 脚手架生成 h5 项目 boilerplate 

####此样本项目包含 mutants(插件能力), chanjet-ui.

#### 开发使用到的主要技术:

* Node / Npm
* ES6
* ReactJs
* 手淘Rem适配方案
* Babel
* Webpack
* cordova

了解这些技术将有利于你的开发

#### 命令
* `npm start`   启动项目
* `npm run zip` 打包成嵌入工作圈项目
* `npm test` 执行单元测试
* `npm run test-coverage` 生成覆盖率报告
